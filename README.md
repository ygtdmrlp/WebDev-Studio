# 🚀 WebDev Studio - Modern HTML Önizleme ve Düzenleme Sitesi

WebDev Studio, modern web geliştiricileri için tasarlanmış, tamamen tarayıcı üzerinde çalışan, hızlı ve kullanıcı dostu bir kod düzenleme ve canlı önizleme aracıdır. Herhangi bir kurulum gerektirmeden HTML, CSS ve JavaScript projelerinizi anında oluşturabilir, düzenleyebilir ve önizleyebilirsiniz.
- Demo Link [demo link](https://webdev-studio.onrender.com/)
![WebDev Studio Banner](https://i.hizliresim.com/906a4be.jpg)

## 🌟 Özellikler

- **⚡ Canlı Önizleme**: Kod yazarken anında sağ panelde değişimleri görün.
- **🖥️ Monaco Editor**: VS Code'un güçlü düzenleyicisi ile profesyonel kod yazma deneyimi (Syntax highlighting, Auto-complete).
- **📂 Yerel Depolama**: Projeleriniz tarayıcıda (IndexedDB) güvenli bir şekilde saklanır, sayfa yenilense bile kaybolmaz.
- **📁 Dosya Yönetimi**: Yeni dosya oluşturma, silme, yeniden adlandırma ve dosya listesi yönetimi.
- **📤 İçeri/Dışarı Aktar**: Mevcut HTML dosyalarınızı yükleyebilir veya projelerinizi tek tıkla indirebilirsiniz.
- **🎨 Modern Arayüz**: Tailwind CSS ile hazırlanmış, Dark/Light mode destekli, responsive tasarım.
- **📱 PWA Desteği**: Uygulamayı masaüstüne veya mobil cihazınıza yükleyebilir, offline olarak kullanabilirsiniz.
- **📝 Hazır Şablonlar**: Landing Page gibi hazır şablonlarla projelerinize hızlı başlayın.

## 🚀 Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/ygtdmrlp/WebDev-Studio.git
   ```

2. Proje dizinine gidin:
   ```bash
   cd WebDev-Studio
   ```

3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

4. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

5. Tarayıcınızda açın: `http://localhost:5173`

## 📦 Build (Üretim Sürümü)

Uygulamayı canlıya almak için:
```bash
npm run build
```
Oluşturulan `dist` klasörünü herhangi bir statik hosting servisinde (Vercel, Netlify, GitHub Pages vb.) yayınlayabilirsiniz.
